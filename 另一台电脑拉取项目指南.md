# 另一台电脑拉取项目指南

## 仓库信息
- 仓库地址: https://github.com/2082743849-beep/gongtu-project.git
- 仓库是公开的，clone 不需要登录任何账号

## 操作步骤

### 1. clone 项目
```bash
git clone https://github.com/2082743849-beep/gongtu-project.git
cd gongtu-project
```

### 2. 切换到当前工作分支
```bash
git checkout feature/csg-v2-integration
```

### 3. 查看所有分支
```bash
git branch -a
```

### 4. 查看所有冻结标签（回滚点）
```bash
git tag -l
```

### 5. 如果要回到某个冻结版本
```bash
git checkout csg-section-v6-interactive
```

### 6. 启动本地预览
```bash
python3 -m http.server 8089
# 浏览器打开 http://localhost:8089/csg-section.html
```

## 注意事项
- clone 下来的代码是公开的，任何电脑都能拉
- 如果要在另一台电脑上 push 修改，需要配置 git 用户信息:
  ```bash
  git config user.name "你的名字"
  git config user.email "你的邮箱"
  ```
- 如果用别人的 Codex 账号，Codex 的 git 身份可能是别人的名字，push 的 commit 作者会显示别人的名字
