# 公途首页中段「内容驱动的国风视觉」Goal

## 权威入口

本文与 `.xds/goals/homepage-content-led-guofeng-redesign.json` 是本轮首页中段整改的唯一权威入口。它是 `unified-v3-experience-repair` 中 V3R-6 的视觉子 Goal，不替代已经完成的正式功能、数据合同、首屏和尾屏成果。

## 当前事实

- 首屏和尾屏已经搬运完成，本轮不再重复搬运。
- 目前中段把一张水墨氛围图当作主要表达，只有风格，没有清楚回答“这一段在讲公途的什么”。
- 用户给出的录屏和图片是审美参照：可学习国风笔触、墨色流动、灰蓝与柔金配色、雾感、留白、精细线条和光影层次；不可照搬其中的人物、武器、桥亭、地图、路线节点等具体元素。
- 用户认可“内容为主、视觉要美、大胆改”，希望前段可有唯美循环氛围，中段保留并优化泼墨与粒子效果，但不能一张图铺到底。
- 中段真正要表达的是公途的学习闭环：题目现场被保存，方法过程可追溯，AI 继续追问，错误形成记忆，并转化为下一次训练。

## 目标

重新设计并落地首页中段，使每一段视觉都由对应产品内容决定，并用统一但不重复的国风视觉语言串联“学习现场 → 证据沉淀 → AI 追问 → 错误记忆 → 下一次训练”。前段使用可循环的氛围动图或等价轻量动效建立情绪，中段通过泼墨显影与粒子反馈呈现证据连续，正文和真实产品信息始终保持第一层级；最终在桌面与手机真实页面中验收。

## 视觉原则

1. 参考的是笔触、墨色、配色、光影、材质和留白，不复制参考图里的具体物件与构图。
2. 先确定这一屏的产品含义、文字层级和用户阅读动作，再决定图的主题、位置、明暗和运动方式。
3. 不用古风人物、桥亭、地图、路线节点、竖牌、长卷或大面积试卷充当无关装饰。
4. 不用一张图片从头铺到尾；不同段落使用同一视觉语言下的不同内容画面、局部纹理和动效节奏。
5. 粒子和泼墨负责揭示关系与状态变化，不遮挡正文，不伪装成可操作功能。
6. 正式产品感优先：避免可读伪文字、素材拼贴感、模板化卡片和廉价发光效果。

## 非范围

- 不改动已完成的首屏、尾屏、认证、题库、AI 上下文、学习记录或用户隔离逻辑。
- 不复刻用户参考视频或图片的具体人物、建筑、地图和镜头构图。
- 不引入仅为炫技的重型 3D/Spark 运行时。
- 不在未获得用户实际看页确认前完成 V3R-6 或主 Goal。

## 阶段计划

| ID | 任务 | 验收条件 |
| --- | --- | --- |
| HCG-1 | 盘点当前页面与参考，建立内容—视觉—动效对应表 | 明确当前中段各屏的产品含义、保留项、移除项、视觉主题和动效职责；能解释每张图为什么出现在该位置。 |
| HCG-2 | 确定统一国风母题与分段美术方案 | 形成桌面/手机分段方案；只借鉴笔触、色彩、光影与留白，明确排除人物、武器、桥亭、地图、路线节点、竖牌和长卷。 |
| HCG-3 | 生成并筛选内容专属视觉素材 | 素材分别服务学习现场、证据链与下一次训练；无可读伪文字、无关古风摆件或参考元素复刻；放入页面前先完成独立视觉检查。 |
| HCG-4 | 重构中段版式和叙事节奏 | 首尾屏保持不动；中段不是一图到底；标题、正文、产品证据和视觉主次清楚，滚动时能读懂五步学习闭环。 |
| HCG-5 | 实现循环氛围、泼墨转场和粒子反馈 | 前段氛围可自然循环；泼墨与章节显影关联；粒子用于证据汇聚/流转；不遮挡内容，并支持 `prefers-reduced-motion`。 |
| HCG-6 | 完成桌面与手机真实验收和修正 | 1280×720 与 390×844 关键截图通过；无横向溢出、破图和新增阻塞错误；内容可读、动效节奏合理。 |
| HCG-7 | 更新状态、证据与交付入口 | `CURRENT_STATUS.md`、`TASKS.md` 和验收记录反映真实结果；记录预览 URL、截图、检查结果、已知边界与用户确认门禁。 |

## 证据要求

- HCG-1：页面结构盘点、参考帧与内容—视觉—动效对应表文件。
- HCG-2：分段方案或设计说明，明确参考与禁止复刻边界。
- HCG-3：项目内素材文件及放入页面前的独立查看记录。
- HCG-4：HTML/CSS/JS diff 与真实页面截图。
- HCG-5：桌面录制或连续截图、减少动态效果检查、相关静态/语法检查。
- HCG-6：桌面和手机截图、页面尺寸指标、破图和溢出检查结果。
- HCG-7：状态文件、验收文档与可访问的本地预览入口。

## 停止条件

- 新方案主要靠参考图中的人物、建筑、地图等元素成立时，停止实现并回到 HCG-2。
- 无法解释某张图与当前段落产品内容的关系时，该素材不得进入页面。
- 动效遮挡正文、引起明显卡顿、破坏手机阅读或不能在减少动态效果下安全降级时，不得进入验收。
- 发现改动触及首尾屏或既有正式产品逻辑时，先记录边界，不继续扩大范围。

## 交付边界

本 Goal 的交付物是首页中段的视觉与动效整改、对应素材、桌面/手机验收证据和用户确认入口。它不包含主 Goal 的合并、PR 审阅或其他模块改版。

## 完成审计

完成前逐项确认：每屏都有明确产品含义；参考只作用于视觉语言；中段不是一张图铺到底；循环氛围、泼墨和粒子各有职责；内容始终可读；桌面与手机真实验收通过；用户已看到页面并确认方向。缺少任一项，本 Goal 继续保持进行中。

## 开 Goal 提示

从 `.xds/goals/homepage-content-led-guofeng-redesign.json` 的 `currentTaskId` 继续，一次只推进一个任务。先完成内容—视觉—动效对应表，再生成图片；每项只能凭文件、测试、URL、截图或用户明确确认完成验证。
