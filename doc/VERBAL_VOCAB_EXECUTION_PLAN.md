# 成语词库与言语题库执行计划

更新时间：2026-07-05
当前分支：`cx/vocab-verbal-original-ui`
工作区：`/Users/miduoduo/Documents/xixi_gongtu-main-app`

## Goal 建议文案

把公途主站的成语内置词库升级为带人民网例句的高频 800 词库，并新增“言语理解 / 逻辑填空”题库模块。要求词库、刷题、收藏、错题和复习状态分层清楚，不破坏现有前端风格，不覆盖历史数据库数据，后续能继续加入“夸夸刷”等其他题库来源。

## 设计原则

1. 不重做前端风格：保留现有浅色学习面板、宣纸感、金色点缀、圆角卡片、顶部牌组切换、中央学习卡、右侧统计/计划/热力图/卡片库、既有 CSS class 和既有图标气质；静态 demo 只表达位置和逻辑，不能把 demo 的新视觉搬进真实页面。
2. 词库和题库拆开：词库负责词条解释、例句、收藏、复习；题库负责题目、答案、解析、作答记录、错题。
3. 题库来源可扩展：`花生海海刷`、后续 `夸夸刷` 都作为 `question_bank` 来源，不另起系统。
4. 同题可归并：同一道题如果出现在多个题库，只保留一个规范题目，记录多个来源。
5. 用户数据不覆盖：任何迁移先备份数据库；内置词库更新不删除用户历史学习记录。

## 数据事实

- `高频成语800词完整版（修复版）.xlsx`
  - 805 行，801 个唯一成语，4 个重复。
  - 列：`成语`、`解释`、`例句`、`人民网搜索`。
  - `例句` 列多数单元格内含换行，导入和显示必须保留。
- `花生海海刷言语理解题库.xlsx/json`
  - 450 题：逻辑填空 231 题，片段阅读 219 题。
  - JSON 与 Excel 答案一致。
  - 已按答案汇总 PDF 进行模块内顺序比对，450 题答案 0 个不一致。
  - Excel 的 `题号` 列存在导出问题，10/20/30 变成 `0`，11-19 显示为 1-9；导入时必须按模块内顺序重建题号。

## 阶段计划

### P0：保护现场与建校验脚本

交付：
- 数据库备份脚本或手动备份记录。
- 只读导入校验脚本：读取 Excel/JSON/PDF，输出词库重复项、题库数量、答案比对结果。

验收：
- `backend/data.db`、`data.db-wal`、`data.db-shm` 已备份。
- 校验报告显示 450 题答案与 PDF 汇总 0 个不一致。
- 报告明确指出 `题号` 列不可直接使用，导入使用模块内顺序号。

### P1：词库数据模型升级

交付：
- `highfreq_vocab` 增加或迁移到支持 `examples`、`source`、`search_url` 的结构。
- 导入 800 词内置词库，保留例句换行。
- 保留用户 `custom_vocab` 和 `user_decks` 历史数据。
- 原“成语学习卡片”继续保留现有前端风格，但背面从单一释义升级为：解释、人民网例句、人民网搜索入口。

验收：
- 内置词库数量为 801 个唯一词条或明确记录重复处理规则。
- 任意抽查 10 个词条，解释与 Excel 一致。
- 任意抽查 10 个词条，例句换行保持为多行。
- 人民网搜索链接可由词语生成，例如 `https://search.people.cn/s?keyword=<词语>`。
- 登录用户历史牌组仍可读取。
- 进入原成语学习卡片，点击翻面后能看到解释、多条例句和“去人民网搜索例句”入口。

### P2：词库学习逻辑整理

交付：
- 将“词条内容”和“我的学习状态”分离。
- 按常见背单词软件逻辑重整：词库是稳定内容库，今日任务是学习队列，到期复习/易忘词/收藏夹是当前用户状态视图。
- 页面模式原位调整：保留现有模式按钮区域，将 `复习模式 / 全部背诵 / 错题攻克` 重新接线为 `今日任务 / 全部词库 / 易忘攻克`；`到期复习` 和 `收藏夹` 放在右侧卡片库过滤器或统计入口，不新增侧边栏式大改版。
- `全部词库` 只浏览，不清空今日队列。
- `今日任务` 由到期复习 + 今日新词组成。
- 原有“复习模式 / 全部背诵 / 设置学习任务 / 错题攻克”重命名并重新接线，避免切换模式或切换词库时清空学习进度。
- `每日学习计划` 保持原右侧卡片位置，只改变其语义：设置每日目标影响明日/新建任务上限，不直接清空当前已生成队列。

验收：
- 切换内置词库/浏览全部词库不会清空历史学习进度。
- 每日新词数量只影响今日任务，不影响全部词库。
- `没记住` 会进入错题/易忘。
- `收藏` 会进入收藏夹，刷新后保留。
- 登出后再登录，当前用户数据恢复；其他用户不可见。
- 原成语学习入口仍可正常背词：今日任务显示到期复习和新词；全部词库可自由点开看词，不改变今日任务。

### P3：言语题库数据模型与导入

交付：
- 新增题库表：题库来源、题目、选项、答案、解析、题型、来源模块、模块内顺序号。
- 导入 `花生海海刷` 450 题。
- 建立同题去重策略：规范化题干 + 选项 + 答案生成指纹。

验收：
- 题库总数 450。
- 逻辑填空 231，片段阅读 219。
- 每个来源模块数量与 PDF 答案汇总一致。
- 答案按模块内顺序比对 0 个不一致。
- `题号` 显示为重建后的模块内顺序号，不使用 Excel 错位题号。

### P4：逻辑填空刷题入口

交付：
- 在学习面板中增加“言语刷题”入口。
- 支持选择题库来源：花生海海刷；后续可加入夸夸刷。
- 支持只刷逻辑填空、刷片段阅读、按套题训练。
- 做题后保存作答、正确/错误、用时和错题状态。

验收：
- 未登录不能保存作答记录。
- 登录后作答记录写入当前用户。
- 刷新后保留进度。
- A/B 用户作答记录隔离。
- 选择 `只刷逻辑填空` 时不会混入片段阅读。
- 做完后能查看正确答案和解析占位；当前解析为空的题显示“暂无解析，先看正确答案”。

### P5：题中词语悬停与收藏

交付：
- 题干和选项中的词语可悬停。
- 优先匹配内置 800 词库，显示解释和多条例句。
- 词库未收录的词，显示“待补充词条”，可收藏或加入待补充列表。
- 支持一键收藏词语。

验收：
- 已收录词条悬停显示解释、例句、人民网搜索链接。
- 例句按 Excel 单元格内换行分行展示。
- 未收录词条不会报错，能收藏为待补充。
- 收藏刷新后保留。
- 题中词条收藏不影响题目作答记录。

## 前端风格保护线

- 不改现有整体色系、字体气质、圆角卡片、顶部导航、右侧统计面板。
- 不重写真实页面的整体 CSS；优先复用现有 class、现有按钮样式、现有图标，新增样式只服务缺失状态且范围最小。
- 不把“今日任务 / 全部词库 / 易忘词 / 收藏夹”做成新的侧边栏导航；优先放回原模式按钮和右侧卡片库区域。
- 不把真实页面改成静态 demo 的紫色渐变、新卡片层级或新排版；真实实现以 `智学成语-高级版.html` 当前页面为视觉事实源。
- 不把学习页改成新的 landing page。
- 不新增大面积强色块或重营销风 hero。
- 新增“言语刷题”必须像现有成语/数量模块的自然扩展。
- 每个阶段只做局部 UI 增量，浏览器截图确认无明显错位、遮挡、横向溢出。
