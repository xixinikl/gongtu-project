# 三视图做题技巧模板

更新时间：2026-07-05

这个模板用于三视图训练题。录入新题时，不能只放 3D 模型和答案，必须让学生知道“眼睛先看哪、怎么排除”。

## 固定节奏

1. 数：先看题目一共几个黑块、几个白块，数量是边界。
2. 锁：用两个已知视图锁定黑块、缺口或凸出来的块在哪一列、哪一层。
3. 挡：判断主视方向最前面看到的是黑块、白块还是空位。
4. 比：只比较选项最不一样的格子，优先排除把俯视图/左视图直接搬成主视图的选项。
5. 验：选完后用 3D 模型切到主视、左视、俯视，三张图必须同时成立。

## 每题必填

- `teaching.short`：一句短技巧，尽量像老师口播。
- `teaching.steps`：3 到 5 句做题步骤，不写专业定理。
- `teaching.optionFocus`：逐项写 A/B/C/D 的差异和排除抓手。

## 禁止写法

- 不写“根据正交投影定理可知”。
- 不写“由空间直角坐标系推导”。
- 不只写“答案 C”，必须说学生应该如何看出来。

## 当前第一题示例

先数 3 个黑块，再看左视图和俯视图把黑块锁到后排或侧边；主视图看最前面，前面被白块挡住时，后面的黑块不会露出来，所以 C 的 3×3 全白成立。
