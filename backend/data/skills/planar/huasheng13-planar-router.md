## 检查顺序

1. 元素组成相同：优先检查平移、旋转、翻转和遍历。
2. 元素组成相似：检查叠加、去同存异、黑白运算与样式变化。
3. 元素组成凌乱：检查点、线、角、面、素和部分数，再看对称、曲直、开闭等属性。
4. 每个候选规律必须解释整组图，不能只解释最后一格。
