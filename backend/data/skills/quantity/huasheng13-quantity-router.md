## 子题型选择

| 当前题型 | 只启用的方法线索 |
| --- | --- |
| 工程 | 总量赋公倍数、效率关系、工作量守恒 |
| 行程 | 路程=速度×时间、相遇/追及关系、单位统一 |
| 利润 | 成本基准、售价/折扣/利润率关系 |
| 容斥 | 集合关系、重复计数与总数边界 |
| 排列组合/概率 | 分步分类、先约束后计数、分子分母同口径 |
| 年龄/最值/几何等 | 只读取匹配小节，不顺带加载其他公式 |
